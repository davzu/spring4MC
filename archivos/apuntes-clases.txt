
Estereotipos
@Component (engloba todos)
@Repository  => clase que tiene acceso a base de datos
@Service => clase que contiene lógica de negocio
@Controller => clase que interactúa con las páginas web

@Autowired => Spring va a buscar alguna implementación para la interfaz, con esto ya no haríamos new
Ejemplo:

@Repository
public class PersonaDAOImpl implements IPersonaDAO .....

@Autowired
private IPersonaDAO daoPersona; => autowired encontrará la implementación PersonaDAOImpl y lo inyectará a daoPersona

---------

<context:component-scan base-package="com...."></context:component-scan> => escanea todas las clases que estén estereotipadas
